# 文风、降 AI 味与质检

## 事实密度

删去形容词和口号后，段落仍须回答：**谁、做什么、怎么做、到什么程度**。

- 好：`将材料由〔N〕项压减至〔M〕项，新增线上预审。`
- 差：`持续优化服务，推动提质增效。`

## 判断强度（证据不足则降级）

| 级别 | 表述 | 条件 |
| --- | --- | --- |
| 1 | 已启动、正在开展 | 仅部署 |
| 2 | 有序推进 | 有步骤、成效有限 |
| 3 | 取得进展 | 有阶段结果 |
| 4 | 明显成效、形成机制 | 有数据或制度 |
| 5 | 重大突破 | 有权威认定或关键指标 |

**宁可稳妥，不要拔高。**

## 句式与抽象词

- 避免连续 `要……要……要……`、堆砌连接词。
- 抽象词（高质量发展、赋能、机制、闭环等）后须接**具体内容**。
- 万能结尾（谱写新篇章）→ 改回任务、时限、责任。

## Humanizer 要点（成稿第二遍）

成簇出现才改：空泛评价 + 整齐三段 + 无主体 + 无事实 + 万能结尾。

| 表现 | 改法 |
| --- | --- |
| 具有重要意义（无据） | 写具体任务或结果 |
| 推动促进助力连用 | 保留一个动作+对象 |
| 有关方面认为 | 写来源或删除 |
| 下面从三个方面 | 直接写内容 |

成稿前自问：**被退回最可能因为什么像 AI？** → 空则补事实，飘则降调，整齐则打散。

## 评分（满分 100，低于 80 须改）

| 项 | 分 | 低分信号 |
| --- | ---: | --- |
| 事实密度 | 20 | 满篇口号 |
| 文种契合 | 20 | 文种错位 |
| 克制 | 15 | 判断过强 |
| 句式自然 | 15 | 机械排比 |
| 抽象词控制 | 10 | 万能词堆叠 |
| 结构清晰 | 10 | 逻辑空转 |
| 省部级站位 | 5 | 基层口径或假高度 |
| 敏感词合规 | 5 | 命中错词词库或禁用表 |

**低于 80 时优先补：事实密度、敏感词合规（含 CSV 错词检索）。**

## 定稿检查清单

- [ ] 文种、主送、事由正确；请示仅一主送
- [ ] 无编造文件、领导、数据、会议
- [ ] 每段有功能；判断有事实支撑
- [ ] 省部级站位够、不假大空
- [ ] 全文通过错词词库与敏感词表（见 reference-sensitive.md、`错词词库-20260708.csv`）
- [ ] 无聊天腔（当然可以、以下是…）
- [ ] 润色任务已附《修改说明》（若适用）
