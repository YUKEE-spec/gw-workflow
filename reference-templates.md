# 文种选择与格式模板

## 法定公文（摘要）

| 文种 | 结构要点 | 纪律 |
| --- | --- | --- |
| 通知 | 依据+事项+要求+时限 | 对象责任明确 |
| 报告 | 情况+做法+成效+问题+下一步 | 不夹带请示 |
| 请示 | 缘由+依据+请求事项+请批语 | 一文一事、一主送 |
| 函 | 来由+事项+盼复/函复 | 平行、礼貌 |
| 批复 | 引述来文+意见+要求 | 明确可否 |

## 事务文书（摘要）

| 文体 | 结构要点 |
| --- | --- |
| 汇报材料 | 背景→情况→做法→成效→问题→建议 |
| 讲话稿 | 称谓→形势→任务→落实→收束 |
| 调研报告 | 背景→现状→问题→原因→建议 |
| 工作方案 | 目标→任务→步骤→分工→保障 |
| 工作总结 | 总体→做法→成效→不足→下步 |

## 易混辨析

- 请示 vs 报告：有批准请求 → 请示
- 方案 vs 计划：方案重组织实施路径
- 汇报材料 vs 请示：汇报不夹「请予批准」

## 格式要点

- 标题：`机关+关于+事由+的+文种`
- 主送：请示一般单一主送
- 附件：`附件：1. XXX`（名称一般不加标点）
- 上行文尊重事实；下行文明确时限责任

## 常用开头

- 根据式：`根据〔部署〕要求，为〔目的〕，现就…通知如下。`
- 汇报式：`现将〔专项〕有关情况汇报如下。`
- 调研式：`围绕〔主题〕，采取〔方式〕开展调研，现将情况报告如下。`

## 常用结尾

- 通知：`请…抓好落实，并于〔日期〕前报送。`
- 请示：`妥否，请批示。`
- 报告：`特此报告。`
- 方案：`各责任单位按职责分工抓好落实。`

## 小标题

用功能标题，不强行四字对仗：`摸清底数，建好台账` / `聚焦薄弱环节，抓实整改`。
